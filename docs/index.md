## Overview

A Python-based analysis tool for evaluating area conservation in ZDEM (Z-Discrete Element Method) granular simulations using Delaunay triangulation.

## Quick Start

```bash
pip install -r requirements.txt
python main.py
```

## Docker

```bash
docker build -t zdem-area-conservation .
docker run --rm zdem-area-conservation
```
