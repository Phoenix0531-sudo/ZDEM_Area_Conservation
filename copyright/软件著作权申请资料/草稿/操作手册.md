# ZDEM 颗粒分布与面积守恒分析系统 操作手册

## 软件概述

本软件用于对 ZDEM 离散元模拟输出数据进行颗粒覆盖面积守恒性分析。

1）ZDEM 数据文件批量解析：用于读取用户指定的 ZDEM 输出数据目录，自动扫描目录中所有 .dat 文件，解析其中的颗粒坐标、半径和颜色编号信息，并转换为 _particles.txt 标准中间格式。该功能用于将 ZDEM 原始二进制输出转换为可供后续分析使用的结构化数据。

2）颗粒颜色筛选：用于按颜色编号选择目标颗粒组别。用户可指定一种或多种颜色编号（如 7,8），也可选择全部颗粒参与分析。该功能适合对不同粒径组或不同材料类型的颗粒分别进行面积守恒性评估。

3）Delaunay 三角剖分：用于以颗粒坐标为节点生成三角网格。基于 scipy.spatial.Delaunay 实现，支持重复点检测与异常容错处理。三角网格是计算颗粒体系有效覆盖面积的几何基础。

4）边长阈值过滤：用于剔除三角网格中的异常跨接三角形。以颗粒平均半径的指定倍数（默认 3 倍）作为边长上限，三边长度均超过该阈值的三角形被判定为颗粒间的不合理跨接并从网格中移除。过滤后的三角网格代表颗粒体系的真实覆盖区域。

5）有效覆盖面积计算：用于对边长阈值过滤后保留的三角形计算叉积面积并逐三角形求和，得到当前时间步颗粒体系的有效覆盖面积。面积以平方单位表示，输出时自动进行千位取整以便阅读。

6）三角网格可视化：用于生成各时间步的 Delaunay 三角剖分网格图。支持输出原始三角网格图和边长阈值过滤后的三角网格图，叠加颗粒位置散点，以 PDF 和 SVG 双格式矢量图保存，满足学术论文插图规范。

7）面积趋势分析与可视化：用于以首个时间步面积为基准，计算各后续时间步的面积变化率（百分比），绘制面积变化趋势曲线。趋势图中自动标注安全波动区间（绿色半透明带）和正负阈值参考线（红色虚线），帮助研究者快速判断模拟过程中颗粒覆盖面积是否保持守恒。

## 运行环境

软件可在配置 Python 3.8 及以上版本的桌面环境中运行，依赖 numpy≥1.21.0、scipy≥1.7.0、matplotlib≥3.4.0。操作系统支持 Windows 10/11、macOS 12+ 和 Ubuntu 20.04+。

## 典型使用流程

ZDEM 颗粒分布与面积守恒分析系统的典型使用流程如下，审核人员可通过该流程理解各项功能之间的衔接关系。

1）将 ZDEM 模拟产生的 .dat 文件放入 data/ 目录。
2）在命令行中执行主分析脚本，根据需要指定数据目录、颜色组别和阈值参数。
3）程序自动将 .dat 文件转换为 _particles.txt 中间格式。
4）程序读取中间文件，对每个时间步依次执行三角剖分、边长过滤和面积计算。
5）在控制台查看面积和颗粒数量的变化分析结果。
6）在 data/ 目录下查看生成的三角网格图（PDF/SVG）和面积趋势图（PDF/SVG）。

## 主要功能说明

软件围绕 ZDEM 数据文件批量解析、三角剖分与边长过滤、有效面积计算和趋势可视化组织功能。

### 1. 数据准备

用户将 ZDEM 模拟输出的 .dat 结果文件放入项目 data/ 目录。程序运行时首先扫描 data/ 目录，识别所有 .dat 文件，将每个文件解析为包含颗粒坐标、半径和颜色编号的 _particles.txt 中间文件。

软件不会修改原始 .dat 文件，转换后的中间文件保存在同一 data/ 目录下，文件名与原始 .dat 文件对应。

操作步骤：
1）在项目根目录确认 data 文件夹已存在。如不存在，手动创建 data 文件夹。
2）将 ZDEM 模拟产生的所有 .dat 文件复制到 data/ 目录中。
3）确认 .dat 文件已正确放置。文件名建议保留 ZDEM 默认命名格式（如 all_0000101000.dat），程序按文件名中的数字编号识别时间步顺序。

完成数据准备后，即可在命令行中启动分析程序。若目录中没有 .dat 文件，程序会在控制台输出提示信息。

### 2. 启动分析

用户在命令行终端（命令提示符或 PowerShell）中进入项目根目录，执行 python Area_Conservation.py 启动分析程序。程序启动后依次完成以下工作：确认数据目录和筛选条件、将 .dat 转换为中间格式、对每个时间步执行 Delaunay 三角剖分、边长阈值过滤和面积计算、在控制台输出统计结果、生成三角网格图和面积趋势图。

程序支持通过命令行参数自定义分析行为，包括数据目录、颗粒颜色组别、边长阈值因子、面积趋势图类型和论文风格输出等。

操作步骤：
1）打开命令行终端（命令提示符或 PowerShell）。
2）进入项目根目录（使用 cd 命令）。
3）使用默认参数执行分析：python Area_Conservation.py
4）或指定参数执行：python Area_Conservation.py --colors=7,8 --threshold=3.5
5）等待程序运行完毕，观察控制台输出的统计结果。

程序运行过程中会逐时间步显示处理进度和面积计算结果。运行结束后，data/ 目录下会生成三角网格图和面积趋势图的 PDF 及 SVG 文件。

【截图预留：命令行终端截图，展示 python Area_Conservation.py 的完整执行过程和输出结果。】

### 3. 查看三角网格图

每个时间步的分析结果会生成两张三角网格图：原始三角网格图（保存在 data/1_triangulation_plot/ 目录）和边长阈值过滤后的三角网格图（保存在 data/2_filtered_triangulation_plot/ 目录）。

三角网格图以绿色散点表示颗粒位置，绿色细线表示 Delaunay 三角网格边。用户可以对比过滤前后的网格变化，判断边长阈值因子设置是否合理。过滤后网格图中不应出现明显不合理的跨接三角形。

操作步骤：
1）在文件资源管理器中进入 data/1_triangulation_plot/ 目录。
2）双击 .pdf 或 .svg 文件，使用 PDF 阅读器或浏览器打开。
3）观察颗粒分布和三角网格连接情况。
4）进入 data/2_filtered_triangulation_plot/ 目录，查看过滤后的网格效果。
5）对比过滤前后的网格差异。如果过滤后三角形过少或剩余三角形连接明显不合理，可调整 --threshold 参数后重新运行分析。

图表以 PDF 和 SVG 矢量格式保存，放大不失真，可直接用于学术论文插图。

【截图预留：PDF 阅读器截图，展示三角网格图中的颗粒散点和三角网格边线。】

### 4. 分析面积趋势

面积趋势图（area_trend.pdf / area_trend.svg）保存在 data/ 目录下，用于观察颗粒体系覆盖面积随 ZDEM 模拟时间步的变化情况。

趋势图的 X 轴为归一化时间步（0 到 1），Y 轴为面积相对变化百分比。图中包含蓝色面积波动曲线（带圆点数据标记）、灰色零参考线、红色正负阈值虚线（默认 ±5%）和绿色半透明安全波动区间。

用户通过观察曲线是否长期超出红色阈值线来判断模拟的物理合理性。若曲线长时间超出 ±5% 的阈值范围，说明模拟过程中可能存在不均匀变形或面积不守恒问题；若曲线在绿色安全区附近小幅波动，说明模拟结果的物理合理性较好。

操作步骤：
1）在文件资源管理器中进入 data/ 目录。
2）双击 area_trend.pdf，使用 PDF 阅读器打开。
3）观察蓝色面积波动曲线的整体趋势。
4）检查曲线是否超出红色 ±5% 阈值虚线。
5）关注曲线在绿色安全区内的小幅波动范围。

若需调整阈值百分比或图表展示模式，可通过 --threshold-percent 和 --plot-area-type 参数重新运行分析。

【截图预留：PDF 阅读器截图，展示面积趋势曲线、红色阈值线和绿色安全波动区间。】

### 5. 论文风格图表输出

软件提供论文风格输出模式，用于生成符合学术期刊插图规范的图表。启用该模式后，坐标轴线加粗至 1.2pt，字号统一为 9-10pt，图例带黑色细边框，适合直接插入 LaTeX 或 Word 论文文档。

操作步骤：
1）在命令行中进入项目根目录。
2）执行：python Area_Conservation.py --paper-style=true --plot-area-type=percentage --threshold-percent=5.0
3）等待程序运行完成。
4）在 data/ 目录中查看生成的 PDF/SVG 图表。

生成的图表在视觉风格上更接近学术出版标准，可与默认风格图表对比使用。用户可根据投稿期刊的具体要求选择是否启用该模式。

## 结果查看与注意事项

分析过程中程序在控制台输出以下统计信息：
- 数据目录和筛选条件确认
- 各时间步文件名、颗粒数量和有效覆盖面积
- 相邻时间步的面积和颗粒数量变化率（百分比）
- 首尾时间步的总体变化率汇总

三角网格图和面积趋势图保存到 data/ 目录下的对应子目录中。图表以 PDF 和 SVG 双格式保存，PDF 适合打印和文档嵌入，SVG 适合网页展示和矢量编辑。

注意事项：
1）首次运行前请确保 data/ 目录中存在至少一个 .dat 文件，否则程序提示目录不存在。
2）默认边长阈值因子为 3.0。若过滤后三角形过少，建议增大阈值因子（如调整为 5.0 或更大）。
3）模拟初期颗粒可能发生重新排列，面积波动较大属于正常现象。应结合模拟物理场景综合判断面积守恒性。
4）图表中的中文字体默认使用 Times New Roman 和微软雅黑组合。如出现中文显示异常，请安装相应中文字体并重新运行。

## 命令行常用参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| --dir | str | data | 指定 .dat 文件所在目录 |
| --colors | str | all | 颗粒颜色编号（逗号分隔，或 all/* 表示全部） |
| --threshold | float | 3.0 | 三角网格边长阈值因子（相对于平均半径的倍数） |
| --threshold-percent | float | 5.0 | 面积趋势图阈值百分比 |
| --hint-band-percent | float | 同阈值 | 安全波动区间宽度百分比 |
| --plot-original-tri | bool | true | 是否绘制原始三角网格图 |
| --plot-filtered-tri | bool | true | 是否绘制过滤后三角网格图 |
| --plot-area-type | str | raw | 面积趋势图类型：raw / percentage / normalized |
| --paper-style | bool | false | 启用论文风格（加粗坐标轴、统一字号） |
| --xmax | float | 70000 | 绘图区域 X 轴最大值 |
| --ymax | float | 25000 | 绘图区域 Y 轴最大值 |

## 命令行常用命令示例

1）使用默认参数执行分析：
python Area_Conservation.py

2）指定颜色组别和边长阈值：
python Area_Conservation.py --colors=7,8 --threshold=3.5

3）百分比模式 + 论文风格输出：
python Area_Conservation.py --paper-style=true --plot-area-type=percentage --threshold-percent=5.0

4）指定自定义数据目录：
python Area_Conservation.py --dir="D:\ZDEM_Results\case_01"

5）原始面积值模式 + 论文风格：
python Area_Conservation.py --paper-style=true --plot-area-type=raw

6）归一化面积模式：
python Area_Conservation.py --plot-area-type=normalized

7）指定绘图区域范围：
python Area_Conservation.py --xmax=50000 --ymax=20000

8）完整参数组合示例：
python Area_Conservation.py --dir=data --colors=all --threshold=3.0 --plot-area-type=percentage --threshold-percent=5.0 --paper-style=true

完成分析后，用户可在 data/ 目录及其子目录中查看生成的三角网格图和面积趋势图。
