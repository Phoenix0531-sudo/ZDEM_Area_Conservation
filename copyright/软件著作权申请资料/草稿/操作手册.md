## 二、说明

ZDEM 颗粒分布与面积守恒分析系统 V1.0适用于科学研究/数值模拟/离散元方法场景。用户进入系统后，可以围绕实际工作内容完成账号进入、业务创建、过程查看、结果确认和资料管理等操作。

日常使用时，岩土工程研究人员、颗粒材料力学研究者、离散元数值模拟工程师、相关专业研究生可以按照页面提示从入口进入相应页面，查看当前业务状态，并根据页面中的按钮、输入框、列表或弹窗继续处理。将 ZDEM 模拟中分散的.dat结果文件自动批量处理，通过 Delaunay 三角剖分和边长阈值过滤算法计算颗粒体系的有效覆盖面积，并以学术论文规范的矢量图表展示面积变化趋势，帮助研究者快速判断模拟的物理合理性。

面向 ZDEM 离散元数值模拟研究的颗粒覆盖面积守恒性分析命令行工具，自动完成从原始.dat文件到面积趋势可视化的完整分析流程。

ZDEM 颗粒分布与面积守恒分析系统是一款面向离散元数值模拟研究的科学计算工具。在 ZDEM 模拟中，颗粒体系在加载过程中的面积守恒性直接反映模拟的物理合理性，但原始 ZDEM 输出为按时间步分片的.dat二进制格式，研究者无法直接获取颗粒覆盖面积随加载进程的变化趋势。本系统自动完成从.dat文件解析到面积趋势可视化的完整分析链路，帮助研究者快速判断模拟结果的可靠性。

本手册用于说明软件的用途、功能特点、运行要求和页面操作流程。各功能章节按用户能够看到的页面、入口、按钮、输入项、提示信息和处理结果进行说明。

## 三、功能特点

用户可在准备数据页面将 ZDEM 模拟输出的.dat文件放入指定目录，供程序批量读取和转换。页面上的data/目录和.dat文件列表会集中呈现当前可操作内容，用户处理完成后可以看到程序启动后会在控制台显示找到的.dat文件数量及文件名列表和文件格式不正确时控制台会输出警告信息。

在运行分析页面中，用户主要处理执行颗粒覆盖面积守恒性分析的主流程，包括数据转换、三角剖分和面积计算。系统把命令行终端窗口、程序运行进度和结果文本、各时间步的面积和颗粒数量统计表、相邻步变化率分析结果和总体变化率汇总放在当前操作区域，处理结束后会反馈程序按顺序显示：数据目录与筛选条件确认信息、各时间步文件名、颗粒数量和有效覆盖面积表格和相邻文件步的面积和颗粒数量变化率。

查看三角网格图页面关注的是查看每个时间步的原始和过滤后 Delaunay 三角剖分结果，验证网格生成和边长过滤效果。用户通过颗粒位置散点（绿色圆点）、三角网格边线（绿色细线）和坐标轴与刻度标签确认当前状态，并在操作结束后获得PDF/SVG文件以矢量格式显示，放大不模糊，可直接用于学术论文插图和过滤后网格图应无明显不合理跨接三角形。

用户可在分析面积趋势页面通过面积变化趋势图判断模拟过程中颗粒覆盖面积是否守恒，识别异常时间步。页面上的面积变化曲线（蓝色折线带圆点标记）、零参考线（灰色实线）、正负阈值线（红色虚线）、安全波动区间（绿色半透明带）和图例说明会集中呈现当前可操作内容，用户处理完成后可以看到绿色安全区间提供直观的面积守恒性判断基准和通过红色阈值线快速发现异常波动时间步。

在论文风格输出页面中，用户主要处理生成符合学术期刊插图规范的论文级图表，满足投稿格式要求。系统把加粗的坐标轴线、统一的10pt字号和优化的图例样式（黑色边框）放在当前操作区域，处理结束后会反馈生成图表的坐标轴线加粗至1.2pt，字号统一为9-10pt、图例带黑色细边框，无半透明背景效果和适合直接插入 LaTeX 或 Word 论文文档。

## 五、准备数据

用户可在准备数据页面将 ZDEM 模拟输出的.dat文件放入指定目录，供程序批量读取和转换。用户在操作系统文件资源管理器中操作文件。

用户完成 ZDEM 数值模拟后，从模拟输出目录中将生成的系列.dat结果文件复制到本项目的data/目录下。

页面会按照data/目录必须存在、data/目录中至少包含1个.dat文件和.dat文件须为 ZDEM 标准输出格式等规则限制或提示用户。处理结束后，用户可以看到程序启动后会在控制台显示找到的.dat文件数量及文件名列表和文件格式不正确时控制台会输出警告信息。

## 六、运行分析

用户可在运行分析页面执行颗粒覆盖面积守恒性分析的主流程，包括数据转换、三角剖分和面积计算。用户打开命令行终端（命令提示符或PowerShell），进入项目根目录，输入 python 命令启动程序。

用户在命令行终端中启动分析程序，根据研究需要指定数据目录、颗粒颜色和阈值参数。

该部分提供命令行终端窗口、程序运行进度和结果文本、各时间步的面积和颗粒数量统计表、相邻步变化率分析结果和总体变化率汇总等页面内容，用户可据此查看状态、填写信息或选择下一步操作。

在该页面中，用户先打开命令行终端，随后进入项目根目录（cd命令），接着输入 python Area_Conservation.py 并回车（使用默认参数运行），之后或输入 python Area_Conservation.py --colors=7,8 --threshold=3.5 指定颜色和阈值，最后等待程序执行完毕。

如果不满足Python 3.8+环境已安装且依赖包齐全、data/目录下存在.dat文件和命令行参数格式正确，颜色编号为整数等要求，用户需要根据页面提示调整后再继续。页面反馈通常包括程序按顺序显示：数据目录与筛选条件确认信息、各时间步文件名、颗粒数量和有效覆盖面积表格、相邻文件步的面积和颗粒数量变化率、首尾时间步的总体变化率汇总和图表保存路径提示。

## 八、分析面积趋势

用户可在分析面积趋势页面通过面积变化趋势图判断模拟过程中颗粒覆盖面积是否守恒，识别异常时间步。用户在data/目录中打开area_trend.pdf或area_trend.svg文件。

用户打开面积趋势图，观察面积变化曲线是否在绿色安全波动区间内，判断 ZDEM 模拟的物理合理性。

用户在分析面积趋势页面会看到面积变化曲线（蓝色折线带圆点标记）、零参考线（灰色实线）、正负阈值线（红色虚线）、安全波动区间（绿色半透明带）和图例说明等信息，并可依据页面显示继续处理。

使用该功能时，用户先在文件资源管理器中进入data/目录，随后双击area_trend.pdf用 PDF 阅读器打开，接着观察蓝色面积波动曲线：X轴为归一化时间步（0到1），Y轴为面积相对变化百分比，之后判断曲线是否长期超出±阈值范围（红色虚线）：超出则可能存在面积不守恒问题，最后关注曲线是否稳定在绿色安全区附近小幅波动：是则说明模拟物理合理性较好。

页面会按照面积趋势图已经生成和推荐使用 PDF 阅读器打开以保持矢量格式等规则限制或提示用户。处理结束后，用户可以看到绿色安全区间提供直观的面积守恒性判断基准和通过红色阈值线快速发现异常波动时间步。

## 十一、常见问题解答

问题：程序提示找不到.dat文件怎么办？
解决方法：请确认data/目录下存在至少一个.dat文件。默认数据目录为项目根目录下的data/文件夹，可通过--dir参数指定其他包含.dat文件的目录。

问题：为什么过滤后一个三角形都没有了？
解决方法：可能是边长阈值因子设置过小，导致所有三角形边长均超过阈值被过滤。尝试增大--threshold参数值，例如从3.0调整为5.0或更大。

问题：面积变化率超过阈值是否意味着模拟结果有问题？
解决方法：不一定。模拟初期颗粒可能发生重新排列导致面积波动较大，这是正常物理过程。需要结合加载阶段和模拟场景综合判断，如果后期面积仍大幅偏离初始值，则需要排查模拟参数设置。

问题：生成的图表中文显示为方框或乱码？
解决方法：图表使用 UTF-8 编码。如中文显示异常，请安装中文字体（如宋体 SimSun 或微软雅黑），推荐使用Adobe Acrobat Reader打开 PDF 文件以获得最佳显示效果。默认已配置Times New Roman和微软雅黑作为后备字体。

## 十二、术语表

| 术语 | 解释 |
| --- | --- |
| ZDEM | Z-离散元方法（Z-Discrete Element Method），一种用于模拟颗粒材料力学行为的数值计算方法 |
| Delaunay三角剖分 | 将离散点集连接为三角形网络的算法，确保任意三角形的外接圆内不包含其他点，生成的网格质量较高 |
| 边长阈值 | 以颗粒平均半径的指定倍数（默认3倍）设定的三角形边长上限，超过此长度的三角形边视为两个不相邻颗粒间的异常跨接 |
| 面积守恒性 | 在理想等体积变形条件下，颗粒体系在水平面上的投影覆盖面积应保持恒定或小范围波动，用于检验离散元模拟的物理合理性 |
| 安全波动区间 | 面积趋势图中以绿色半透明带标注的允许面积波动范围，默认±5%，在该区间内的面积变化视为正常 |
