# 业务理解

- 软件名称：ZDEM 颗粒分布与面积守恒分析系统
- 产品定位：面向ZDEM离散元数值模拟研究的颗粒覆盖面积守恒性分析命令行工具，自动完成从原始.dat文件到面积趋势可视化的完整分析流程
- 面向领域 / 行业：科学研究/数值模拟/离散元方法
- 核心价值：将ZDEM模拟中分散的.dat结果文件自动批量处理，通过Delaunay三角剖分和边长阈值过滤算法计算颗粒体系的有效覆盖面积，并以学术论文规范的矢量图表展示面积变化趋势，帮助研究者快速判断模拟的物理合理性
- 证据文件：`业务理解证据.md`

## 目标用户

- 岩土工程研究人员
- 颗粒材料力学研究者
- 离散元数值模拟工程师
- 相关专业研究生

## 主要业务功能

- ZDEM结果文件批量解析
- 颗粒颜色筛选
- Delaunay三角剖分
- 边长阈值过滤
- 有效覆盖面积计算
- 三角网格可视化
- 面积趋势分析

## 功能说明

- ZDEM结果文件批量解析：自动扫描指定目录下所有ZDEM .dat文件，解析其中的墙体、颗粒、接触和粘结数据，转换为包含颗粒坐标、半径和颜色编号的中间文件
- 颗粒颜色筛选：支持按颜色编号筛选特定类型颗粒（如不同粒径组），或选择全部颗粒参与分析
- Delaunay三角剖分：基于scipy.spatial实现Delaunay三角剖分，以颗粒坐标为节点生成三角网格
- 边长阈值过滤：以颗粒平均半径的指定倍数（默认3倍）为边长阈值，剔除三边均超标的异常三角形，消除颗粒间的不合理跨接
- 有效覆盖面积计算：对过滤后保留的三角形计算叉积面积并求和，得到该时间步颗粒体系的有效覆盖面积
- 三角网格可视化：生成原始三角网格和过滤后三角网格的PDF+SVG矢量图，叠加颗粒位置，支持自定义绘图区域
- 面积趋势分析：以首个时间步面积为基准，计算各步面积变化率，绘制趋势曲线并在图中标注安全波动区间和阈值线

## 典型操作流程

1. 用户将ZDEM模拟产生的.dat文件放入data/目录
2. 在命令行执行主分析脚本，指定数据目录、颜色筛选条件和阈值因子等参数
3. 程序自动将.dat文件转换为_particles.txt中间格式
4. 程序读取中间文件，对每个时间步进行Delaunay三角剖分、边长过滤和面积计算
5. 程序在控制台输出面积和颗粒数量的变化分析结果
6. 程序在data/目录下生成三角网格图和面积趋势图的PDF+SVG文件

## 操作手册页面/流程模块

1. 准备数据：用户在操作系统文件资源管理器中操作文件
   - 使用场景：用户完成ZDEM数值模拟后，从模拟输出目录中将生成的系列.dat结果文件复制到本项目的data/目录下
   - 操作要点：在项目根目录确认data文件夹已存在（如不存在则手动创建）；将ZDEM模拟产生的所有.dat文件复制到data/目录中；确认.dat文件已正确放置
2. 运行分析：用户打开命令行终端（命令提示符或PowerShell），进入项目根目录，输入python命令启动程序
   - 使用场景：用户在命令行终端中启动分析程序，根据研究需要指定数据目录、颗粒颜色和阈值参数
   - 操作要点：打开命令行终端；进入项目根目录（cd命令）；输入 python Area_Conservation.py 并回车（使用默认参数运行）；或输入 python Area_Conservation.py --colors=7,8 --threshold=3.5 指定颜色和阈值
3. 查看三角网格图：用户在data/1_triangulation_plot/目录中查看原始三角网格图，在data/2_filtered_triangulation_plot/目录中查看过滤后三角网格图
   - 使用场景：用户分析完成后，打开生成的三角网格图查看颗粒空间分布和三角剖分质量，确认边长阈值参数设置合理
   - 操作要点：在文件资源管理器中进入data/1_triangulation_plot/目录；双击.pdf或.svg文件用PDF阅读器或浏览器打开；观察颗粒分布和三角网格连接情况；进入data/2_filtered_triangulation_plot/目录查看过滤后效果
4. 分析面积趋势：用户在data/目录中打开area_trend.pdf或area_trend.svg文件
   - 使用场景：用户打开面积趋势图，观察面积变化曲线是否在绿色安全波动区间内，判断ZDEM模拟的物理合理性
   - 操作要点：在文件资源管理器中进入data/目录；双击area_trend.pdf用PDF阅读器打开；观察蓝色面积波动曲线：X轴为归一化时间步（0到1），Y轴为面积相对变化百分比；判断曲线是否长期超出±阈值范围（红色虚线）：超出则可能存在面积不守恒问题
5. 论文风格输出：用户在命令行中添加--paper-style=true参数，并可配合--plot-area-type=percentage使用百分比模式
   - 使用场景：用户准备向学术期刊投稿，需要使用符合期刊排版规范的图表插图
   - 操作要点：在命令行中进入项目根目录；输入 python Area_Conservation.py --paper-style=true --plot-area-type=percentage --threshold-percent=5.0；等待程序运行完成；在data/目录中查看生成的PDF/SVG图表
